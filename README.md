# AstroPC

Setup of a Odriod M1 based on the idea of Astroberry (www.astroberry.io)

## Installation

This installation is based on ARMBIAN (www.armbian.com). Please use the user "astrodroid" for the installation.

```
sudo ./install_astro.sh
```

## Post-Installation

### Setup VNC-PWD

```
tigervncpasswd
```

### Reboot

```
sudo reboot
```
